# Paito-SDY
## Tentang Kami
Paito SDY adalah kumpulan data historis hasil pengeluaran Angka SDY yang disajikan dalam bentuk tabel terstruktur. Format tabel ini dirancang secara khusus untuk mempermudah para pembaca dalam melakukan analisis.
